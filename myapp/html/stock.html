Abstract

This Jupyter notebook contains the first part of my implementation of a stock price forecast system using machine learning. To create the system, I used Apple daily closing prices from 2022-2023 to predict prices for January 2024. I first performed Time Series Analysis on the data, considering the prices' stationarity, autocorrelation values, and trend decomposition. I then trained and evaluated several models, before finally using the best performer to create a Shiny application to allow users to forecast the stock prices of any company.

3.2) Stationarity

Stationarity is a measure of the constancy of a time series; a stationary dataset has no seasonal patterns or trends, which could include buying/selling more shares at the beginning/end of each fiscal quarter of before/after holidays.

The Dickey-Fuller test is a common test to determine stationarity, and uses the rolling means and standard deviations of the series. The null hypothesis is "series is not stationary", while the alternative hypothesis is "series is stationary". I set a p-value threshold of 0.05: the test yielded a p-value of 0.265, indicating that the closing prices are NOT stationary.

I first visualized the rolling means and standard deviations, then performed the test.

3.3) Autocorrelation

Autocorrelation is correlation between a time series and its own lag; high autocorrelation values indicate that the two are correlated, i.e. that past values influence future values. The autocorrelation values between the closing prices and its 1, 2, 3, and 4-week lags are all high: buy/sell decisions are made largely based on recent market behavior.

4.1) Helper Function

I first defined a function to plot price forecasts.

In[18]:
def plot_forecast(forecast, model_name, axis):
    
    axis.plot(train.Date[-100:], train.Close[-100:], color="b", label="Training Data")
    axis.plot(valid.Date, valid.Close, color="m", label="True 2023Q4 Prices")
    axis.plot(valid.Date, forecast, color="g", label="Forecasted 2023Q4 Prices")

    axis.set(title=f"{model_name} 2023Q4 Forecast",
        ylim=(160, 200), xticks=axis.get_xticks())
    axis.set_xticklabels(axis.get_xticklabels(), ha="right", rotation=30)
    axis.legend()

    rmse = round(root_mean_squared_error(valid.Close, forecast), 3)
    axis.text(pd.to_datetime("2023-10-10"), 162, f"Forecast RMSE: {rmse}")

4.4) LSTM

The final model is an LSTM neural network, which I chose for its ability to prioritize "recent" data over "older" data when making predictions. I trained the model to predict each day's price using the prices from its past 60 days. This model had by far the best performance.
